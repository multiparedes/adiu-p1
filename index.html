Bienvenido a la práctica uno de ADIU 👋

En esta práctica, nos adentraremos en el mundo de la visualización de datos con el uso de la librería Highcharts. El objetivo principal es aprender cómo representar información de manera efectiva y atractiva.

Para esta exploración, hemos seleccionado la API de Ergast, una fuente de datos completa relacionada con la Fórmula 1. Esta API nos proporciona una amplia variedad de información, desde resultados de carreras hasta detalles sobre pilotos y equipos, clasificaciones, calendarios de eventos y más. Es importante mencionar que en este proyecto hacemos uso de Tailwind CSS, una herramienta que simplifica el desarrollo de interfaces de usuario y ayuda a crear visualizaciones atractivas y funcionales. Además, hemos implementado un sistema de Integración Continua y Despliegue Continuo (CI/CD) para automatizar pruebas y la implementación de código, lo que agiliza el proceso de desarrollo y garantiza una entrega confiable.

Te invitamos a unirte a nosotros en esta exploración donde utilizamos Highcharts, Tailwind CSS y prácticas de CI/CD para analizar datos relacionados con la Fórmula 1 y crear visualizaciones efectivas. ¡Empecemos!
